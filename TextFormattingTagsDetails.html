Tags Used Inside <body> Tag

1. <h1> to <h3> - Headings

Used to define headings of different levels. <h1> is the main heading, and <h3> is a sub-subheading.

2. <p> - Paragraph

This tag is used to write normal text or paragraphs. Each <p> creates a new block of text.

3. <strong> - Strong Importance

This tag highlights text with strong importance. It usually appears bold.

4. <code> - Code Snippet

This tag is used to display HTML tags or programming code in a monospace font. Useful for showing code examples.

5. <hr> - Horizontal Rule

This tag creates a horizontal line across the page. It is used to separate sections.

6. <ul> and <li> - Unordered List and List Items

<ul> creates a bulleted list.
<li> defines each item in the list.
